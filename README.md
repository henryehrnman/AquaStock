# AquaStock - Henry Ehrnman
#
# At a high level I want to create a website and/or app the allows individuals to figure out how to best stock their aquariums with fish, invertebrates, corals, and amphibians. The program will prompt users for their tank size and water parameters then will give all the different options that will work for their setup. It will also give popular suggestions to try to make a proper set up easily obtainable for beginners or bring inspiration to long time hobbyists.
